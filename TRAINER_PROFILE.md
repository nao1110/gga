# トレーナープロフィール機能

## 概要
キャリアコンサルタント（トレーナー）が自分のプロフィールを登録・編集できる機能です。受験者が予約リクエストする際に、トレーナーの情報を見て選びやすくなります。

## 機能詳細

### プロフィール項目

1. **ニックネーム（表示名）**
   - 受験者に表示される名前
   - 本名でもイニシャルでもニックネームでもOK
   - 例：「田中先生」「T.Tanaka」「キャリアサポーター田中」
   - 最大100文字

2. **経歴・専門分野**
   - 業界経験や職種経験を記載
   - 社名は記載不要（業界名・職種名のみでOK）
   - ただし、社名を書きたい場合は書いてもOK
   - フリーフォーマット
   - 最低20文字以上

3. **対応可能時間帯**
   - 面談対応が可能な曜日・時間帯
   - 受験者が予約リクエストしやすくなる
   - フリーフォーマット
   - 最低10文字以上

## 画面構成

### プロフィール表示・編集画面
- **URL**: `/page/trainer/profile.php`
- **アクセス**: トレーナーマイページのヘッダーに「プロフィール」ボタン

### 表示内容
1. **基本情報（表示のみ、編集不可）**
   - 本名（登録名）
   - メールアドレス

2. **編集可能項目**
   - ニックネーム
   - 経歴・専門分野
   - 対応可能時間帯

### プロフィール更新処理
- **URL**: `/page/trainer/profile_update.php`
- **メソッド**: POST
- **処理内容**:
  - バリデーション
  - データベース更新
  - セッション情報更新
  - 成功メッセージ表示

## データベース

### trainersテーブルに追加したカラム

```sql
ALTER TABLE trainers 
ADD COLUMN nickname VARCHAR(100) NULL COMMENT 'ニックネーム（表示名）',
ADD COLUMN career_description TEXT NULL COMMENT '経歴（業界・職種など）',
ADD COLUMN available_time TEXT NULL COMMENT '都合の良い時間帯';
```

## バリデーション

### ニックネーム
- ✅ 必須入力
- ✅ 最大100文字

### 経歴
- ✅ 必須入力
- ✅ 最低20文字以上

### 対応可能時間
- ✅ 必須入力
- ✅ 最低10文字以上

## 使用例

### ニックネーム例
- 「田中 ミサキ」（本名）
- 「田中先生」
- 「M.Tanaka」（イニシャル）
- 「キャリアアドバイザーT」
- 「みさき」

### 経歴例
```
・IT業界で15年のキャリア（SE→プロジェクトマネージャー→人事部）
・大手メーカーで新卒採用や人材育成を担当
・キャリアコンサルタント資格取得後、フリーランスとして活動
・専門分野：IT業界への転職支援、キャリアチェンジ支援
```

### 対応可能時間例
```
・平日：19:00〜22:00
・土日：10:00〜18:00
・基本的に夜間と週末が対応可能です
・事前にご相談いただければ、平日昼間も調整可能です
```

## テスト手順

### 1. データベースマイグレーション実行
```bash
mysql -u root < /Applications/XAMPP/xamppfiles/htdocs/gs_code/gga/database/add_trainer_profile.sql
```

### 2. トレーナーとしてログイン
- **URL**: http://localhost/gs_code/gga/page/trainer/login.php
- **テストアカウント**: tanaka.trainer@example.com / password123

### 3. プロフィールページへアクセス
- マイページのヘッダーにある「プロフィール」ボタンをクリック
- または直接 http://localhost/gs_code/gga/page/trainer/profile.php にアクセス

### 4. プロフィール入力
- ニックネーム、経歴、対応可能時間を入力
- 「プロフィールを保存」ボタンをクリック

### 5. 確認
- 成功メッセージが表示される
- 入力した内容が保存されている
- データベースで確認:
```sql
SELECT id, name, nickname, career_description, available_time 
FROM trainers 
WHERE id = 1;
```
